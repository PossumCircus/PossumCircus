<h4>개발자로서</h4>
<p>처참한 아키텍처를 겪은 후 아키텍처의 중요성을 알고 있습니다.</p>
<p>DX가 UX로 이어진다고 생각합니다.</p>
<p>협업 및 커뮤니케이션 능력이 개발자의 주요 역량임을 압니다.</p>
<p>문서화의 중요성을 압니다.</p>

<h4>개인으로서</h4>
<p>회복 탄력성이 좋은/생산성이 있는 사람이 되고자 합니다.</p>
<p>생각하고, 쓰고, 지우기를 반복하며 글을 쓰는 법을 연습하고 있습니다.</p>
